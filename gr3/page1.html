Principe

Le programme ci-dessus permet de réduire la taille d’une image en ne conservant qu’un pixel sur un certain nombre défini, grâce à la bibliothèque Python PIL (Pillow). En ouvrant une image d’origine, le script calcule les dimensions de la nouvelle image en fonction d’un facteur de réduction (par exemple, 3 pour ne garder qu’un pixel sur 3). Il crée ensuite une image vide aux dimensions réduites, puis parcourt chaque pixel de cette nouvelle image pour y copier la couleur d’un pixel équivalent dans l’image originale
